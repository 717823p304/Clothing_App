CLOTHING APP

A simple and responsive Clothing Shopping Web Application built using React JS.

PROJECT DESCRIPTION

This is a React-based Clothing App that displays different categories of clothes such as:
* Men
* Women
* Children
Users can browse different clothing sections and view products in each category.

FEATURES

1. Category-based navigation (Men, Women, Children)
2. Clean and simple UI
3. Organized folder structure
4. Reusable components
5. Styled using CSS
6. Built with Create React App

TECHNOLOGIES USED

1. React JS
2. JavaScript (ES6)
3. HTML5
4. CSS3
